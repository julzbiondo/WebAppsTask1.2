String Methods

String Length

This method counts each character of a chosen string.

In this case, the string we will count the length of is "Pineapple".

String replace

This method replaces the current text with a new string, for this example we will use a button which will change the text.

In this case, the string we will start with is "Pineapple" and we will change it to "Computer".

String toUpperCase

This method turns the chosen to into all uppercase letters.

For this example we will use the string:

I would be delighted if the sea were full of cucumber juice.

String substring

This method extracts part of a string and returns the extracted parts in a new string

In this example, we will cut 9 characters from the string, starting from 4 characters into the string.

The worst thing about being at the top of the career ladder is that there's a long way to fall.

Number Methods

toString

This method returns the number as a string.

In this example, we will use the interger 43

toExponential

This method returns the number as a string in exponential notation.

Starting number is 5.335

toFixed

This method returns the number rounded to a set decimal place.

The starting number is 53.45284 and it will be rounded to 2 decimal places

toPrecision

This method returns the number written with a specified length

The starting number is 35.298740

This number will be written to the lenghts, 3, 5 and 7

ValueOf

This method returns the number as a number.

The number will be 284

Array Methods

Array length

This method returns the length of an array

In this example, the array will be 5 items long

The array is ["Tree", "Potato", "Green", "Table", "Pineapple"]

Array pop

This method removes the last element of an array and returns that element.

In this example, the array will be ["43", "64", "234", "6345"]

Array flat

This method creates a new array ussing the given array with sub array elements

In this example, the array will be [[23", "342"], ["523432", "1223"], ["23423", "3243"]]

Array concat

This method joins two or more arrays and returns a copy of the joined arrays

In this example, the arrays will be ["Tree", "Potato", "Green", "Table", "Pineapple"] and ["43", "64", "234", "6345"]

Array join

This method joins all elements of an array into a string

In this example, the array will be ["Tree", "Potato", "Green", "Table", "Pineapple"]
